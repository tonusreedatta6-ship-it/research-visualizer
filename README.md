# 📊 Research Visualizer

## 🧠 Project Overview
Research Visualizer is a web-based application that allows users to search any university and visualize academic research data in an interactive way. It displays research distribution across different fields like AI, ML, Networks, and Data Science using bar charts and pie charts.

## 🚀 Features
- Search any university dynamically  
- Interactive bar chart visualization  
- Pie chart representation of data  
- SQLite database integration  
- Flask backend API  
- Streamlit frontend UI  
- Random fallback data support for unmatched universities  

## 🛠️ Tech Stack
- Python  
- Flask  
- Streamlit  
- SQLite  
- Pandas  
- Plotly  

## 📂 Project Structure
backend → Flask backend logic  
frontend → Streamlit user interface  
scripts → Database initialization & data scripts  
data → SQLite database file  

## ▶️ How to Run

1. Clone the repository  
git clone https://github.com/tonusreedatta6-ship-it/research-visualizer.git 

2. Install dependencies  
pip install -r requirements.txt  

3. Run backend  
python -m backend.app  

4. Run frontend (new terminal)  
streamlit run frontend/app.py  

5. Open in browser  
http://localhost:8501  

## 📌 Example
Input: University of Toronto  
Output: Interactive bar and pie charts of research distribution
